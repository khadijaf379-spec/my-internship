# theory
1- arthmetic operators
 JavaScript arithmetic operators take numerical values (literals or variables) as operands and perform standard mathematical calculations. They return a single numerical value.These are arthmetic operators:
 +,-,*,/,%,**exponential

 example 
 let a = 5;
 let b = 2;
 console.log("a + b = ", a + b);
 2- unary operators
 A JavaScript unary operator is an operator that performs an action on exactly one operand (a single variable or value). They are highly efficient and are placed either before (prefix) or after (postfix) the operand
 increament operator = ++ (prefix)
 decreament operator = -- (postfix)
 example: 
 let a = 5;
 let b = 2;
 console.log("a + b = ", a + b);
